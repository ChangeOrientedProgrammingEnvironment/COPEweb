Successful software constantly evolves: change is the heart of software development. We are a team of researchers committed to improving the lives of software developers by understanding the practice of code changes. We have a track record of using this understanding to develop tools that we contribute to the Eclipse and NetBeans development environments.

We have an ongoing study on understanding how developers change code when they follow Test Driven Development (TDD).

By participating in this study you can help to advance the state of the art. We will also send you a personalized report on how your development practices compare with the TDD practices of others.
